# Stela Nubo
Simple Asteroid Shooter Example Game in Rust wit the Bevy engine

## Controls
- Use W,A,S,D to accelerate and strafe
- Q and E Does a roll
- Spacebar to fly up
- Shift to fly down
- Use mouse to turn
- Left mouse button fires cannon



## Goals

- [x] Move around the space in a spaceship

- [x] Input System
- [x] Be able to shoot

- [x] destruction effect
- [x] player damage
